# Clínica Dental Óscar Puertollano

A modern, responsive website for a professional dental clinic located in Portugalete, Bizkaia. This project showcases dental services, professional staff, and provides an easy way for patients to book appointments.

## Description

This website is designed to provide a welcoming and professional online presence for a dental clinic. It features:
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices using Tailwind CSS.
- **Service Showcases:** Detailed sections for various treatments including Implantology, Orthodontics, and Esthetics.
- **Appointment Booking:** A functional-ready contact form for patient inquiries and scheduling.
- **Testimonial Slider:** Dynamic display of patient feedback to build trust.
- **Team Profiles:** Introduces the professional medical staff.
- **Interactive Map:** Integrated Google Maps for easy location finding.

### Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **Tailwind CSS:** Modern utility-first CSS framework for rapid UI development.
- **JavaScript:** Vanilla JS for interactive elements like the mobile menu and testimonial slider.
- **Font Awesome:** For crisp, scalable vector icons.

## Installation

No complex installation is required as this is a static website.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/clinica-dental.git
    ```
2.  **Navigate to the folder:**
    ```bash
    cd clinica-dental
    ```

## Usage

To view the website, simply open the `index.html` file in any modern web browser.

Alternatively, you can use a local development server:
```bash
# Using VS Code Live Server extension or:
npx serve .
```
